# 📁 مجلد التوثيق (MD Files)

## 📋 نظام التسمية

جميع ملفات التوثيق (`.md`) يتم حفظها في هذا المجلد مع بادئة التاريخ والوقت.

### صيغة التسمية:
```
YYYYMMDD_HHMMSS_اسم_الملف.md
```

**مثال:**
```
20260423_204102_DATABASE_MIGRATION_REPORT.md
```

حيث:
- `YYYYMMDD` = التاريخ (سنة شهر يوم)
- `HHMMSS` = الوقت (ساعة دقيقة ثانية)
- `اسم_الملف` = اسم الملف الوصفي
